# iNikah – KUA Karangdadap 💍

Aplikasi web portal informasi pernikahan satu pintu (*one-stop information*) berbasis mobile-first untuk Kantor Urusan Agama (KUA) Kecamatan Karangdadap, Kabupaten Pekalongan.

## ✨ Fitur Utama
* **Tahapan Mandiri Catin:** Alur bimbingan perkawinan (Bimwin), Pre-Test/Post-Test, dan Perpustakaan Digital (Bab 1–10).
* **Cek & Klaim E-Sertifikat:** Sistem pencarian dan unduh lembar kelulusan Bimwin mandiri yang terintegrasi dengan Google Sheets.
* **Layanan Penjadwalan:** Akses *real-time* jadwal akad nikah dan surat penunjukan tugas Penghulu KUA.
* **Informasi & Dokumen:** Unduh mandiri Blangko Dokumen Nikah (N-1 s.d N-6) dan konsultasi interaktif via WhatsApp Penghulu.

## 🛠️ Teknologi
* **Frontend:** HTML5, CSS3 (Custom iOS Native System UI Blur & Gradients), Bootstrap 5.3, Material Icons.
* **Backend & Database:** Google Apps Script (Fetch API) & Google Sheets (sebagai database sertifikat & jadwal).

## 🚀 Cara Penggunaan
Cukup buka file `index.html` di browser Anda, atau *deploy* kode HTML tersebut ke platform hosting gratis seperti GitHub Pages, Netlify, atau Vercel.
