Kebijakan Privasi

Terakhir diperbarui: 24 Juni 2026

Aplikasi iNikah ("Aplikasi") dikembangkan oleh KUA Karangdadap, Kabupaten Pekalongan. Kami menghormati privasi pengguna dan berkomitmen melindungi data pribadi Anda.

1. Data yang Kami Kumpulkan

Aplikasi ini mengumpulkan data berikut saat Anda menggunakan layanan kami:

  • Nama lengkap dan Nomor KTP (saat mengisi Pre-Test dan Post-Test)
  • Alamat email (saat mengisi Post-Test)
  • Jawaban evaluasi Pre-Test dan Post-Test beserta skor
  • Nama lengkap (saat pencarian sertifikat)

2. Penggunaan Data

Data yang dikumpulkan hanya digunakan untuk:

  • Mengevaluasi pemahaman peserta bimbingan perkawinan
  • Menerbitkan e-sertifikat kelulusan Bimwin
  • Menampilkan informasi jadwal akad nikah dan petugas penghulu
  • Mengirimkan notifikasi terkait layanan KUA

3. Penyimpanan Data

Data disimpan di server hosting yang dikelola oleh KUA Karangdadap dengan perlindungan sebagai berikut:

  • Database dilindungi dengan autentikasi dan akses terbatas
  • API dilindungi dengan token autentikasi untuk aksi sensitif
  • Koneksi menggunakan HTTPS (terenkripsi)

Aplikasi juga menyimpan preferensi tema (terang/gelap) di penyimpanan lokal perangkat Anda (localStorage). Data ini tidak dikirim ke server.

4. Layanan Pihak Ketiga

Aplikasi menggunakan layanan pihak ketiga berikut:

  • Cloudflare — untuk CDN dan proteksi keamanan
  • Google Fonts — untuk tampilan huruf (tidak mengumpulkan data pribadi)

Data pribadi Anda tidak dibagikan ke pihak ketiga manapun.

5. Keamanan

Kami menerapkan langkah-langkah keamanan berikut:

  • Koneksi HTTPS untuk seluruh transmisi data
  • Autentikasi berbasis token untuk panel admin dan petugas
  • Proteksi terhadap SQL injection dan XSS
  • Pembatasan akses file dan direktori di server

Namun, tidak ada metode transmisi data melalui internet yang 100% aman.

6. Hak Pengguna

Anda berhak untuk:

  • Tidak menggunakan fitur yang memerlukan input data pribadi
  • Menghapus data localStorage kapan saja melalui pengaturan browser
  • Menghubungi kami untuk pertanyaan terkait privasi

7. Kontak

Jika Anda memiliki pertanyaan tentang kebijakan privasi ini, hubungi kami melalui:

  • WhatsApp: 085226466464
  • Alamat: KUA Karangdadap, Kabupaten Pekalongan

8. Perubahan Kebijakan

Kami dapat memperbarui kebijakan ini sewaktu-waktu. Perubahan akan ditampilkan di halaman ini dengan tanggal pembaruan terbaru.
